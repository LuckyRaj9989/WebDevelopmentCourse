1) Arithmatic Operators:
    +, -, %,*,**,/,++,--

    Increment (++):
        var a = 20;
        a = a+1; ---> This can be written as a++
    Decrement (--):
        var a = 10;
        a = a-1 ---> a--
2) Assignment Operator:
    =,+=,-=,*=,/=,%=,**=

    Addition:
        var a = 55;
        var b = 25;
        a = a + b ----> a += b;
    Subtraction:
        a = a-b ---> a -= b;
3) Comparison Operator:
    >, >=, <, <=, !=, !==, ==, ===

4) Logical Operator:
    &&, ||, !
